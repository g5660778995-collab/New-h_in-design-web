# New H. in Design Web

A fresh prototype direction for H. in Design, inspired by quiet Japanese editorial portfolio systems and the responsive rhythm studied from Shhh.jp.

## Design Direction

- Quiet Japanese editorial minimalism
- Large negative space
- Monochrome base with image-led color
- Slow reveal interaction
- Desktop horizontal gallery flow
- Tablet and mobile vertical magazine flow
- No card-heavy landing page structure

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Connect this repository to Vercel. The default Vite build settings should work:

- Build command: `npm run build`
- Output directory: `dist`

## Current Prototype Scope

This first version is intentionally independent from the existing H. in Design production site. It does not connect to Sanity yet. The goal is to test visual direction, spacing, motion and responsive behavior first.
